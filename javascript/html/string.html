String Methods and Properties

Before starting string methods and properties we need know that strings are indexed. So basically we can access any single character of a string by doing something like this.

Expression: 'Hello World'[1]

Result: 'e'

There is only one property that exists.

Property

  • length

Methods (Without Argument)

  • toUpperCase()
  • toLowerCase()
  • trim()
  • trimStart()
  • trimEnd()

Methods (With Argument)

  • includes() : return boolean
  • indexOf() return number
  • replace() : It takes 2 argument, replace("hi", "hello"), first word present in the sentance and second to replace that word.
  • replaceAll() : replace all the matching words, replaceAll("hi", "hello")
  • concat() : take as many argument to add all the strings.
  • padStart()
  • padEnd()
  • charAt()
  • charCodeAt()
  • split() : return array
